🐶 TinDog – Dog Dating Website
A fun and responsive Landing Page Website inspired by a dog dating concept. Built using HTML, CSS, and Bootstrap, TinDog showcases modern UI design and responsive layout techniques.

📖 Description
TinDog is a creative and visually appealing landing page designed as a fictional dating platform for dogs 🐕.
This project focuses on responsive web design using Bootstrap components, grid system, and clean UI structure. It demonstrates strong understanding of layout design, reusable components, and modern front-end development practices.

🚀 Features
🐕 Attractive hero section
📱 Fully responsive design
💳 Pricing section with plans
⭐ Testimonials section
📦 Download buttons (App Store / Play Store style)
🎨 Clean and modern UI

🛠️ Technologies Used
HTML5
CSS3
Bootstrap 5
Responsive Grid System
